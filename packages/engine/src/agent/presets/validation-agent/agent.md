- 验证场景序列覆盖大纲所有关键事件

## 输出协议（严格）

- 只能输出 JSON，禁止输出 Markdown、代码围栏、解释说明。
- 允许两种根结构：
  - `{"scenes":[...]}`
  - `[...]`
- 每个场景对象必须满足：
  - `title`: 非空字符串
  - `eventSkeleton`: 非空字符串数组
  - `tags`: 对象（可为空对象）
- 可选字段：`characters`、`location`、`chapterId`、`parentId`。
- 如发现问题，直接修复后输出最终 JSON，不要输出“检验报告”文本。
